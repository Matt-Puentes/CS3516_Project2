=================
   The program  
=================
This program is a simulator for layer 4 reliable data transfer.
This was the second project for WPI's CS3516 Computer Networks course

================
   How to run   
================
You can compile this program by running the makefile with "make"
The program can then be run by running ./proj2 and putting in the arguments indevidually.
Alternitavely, the program can be run with the arguments in the command line
EX: ./proj2 10000, 0.9, 0.9, 0.9, 1000, 0, 1, 0
